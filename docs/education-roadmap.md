# Education Roadmap

Goal:

Transition from Backend Engineer to AI Engineer through NeuroAtlas AI.

---

# Stage 1 — Machine Learning Foundations

Duration: 2–3 weeks

## Learn

### Scikit-Learn

Topics:

* Features
* Targets
* Classification
* Regression
* Train/Test Split
* Cross Validation

Resources:

* Scikit-Learn User Guide

---

### XGBoost

Topics:

* Decision Trees
* Gradient Boosting
* Feature Importance

Goal:

Train first medical prediction model.

---

### SHAP

Topics:

* Explainable AI
* Feature Attribution

Goal:

Explain model predictions.

---

# Stage 2 — LLM Fundamentals

Duration: 1–2 weeks

## Learn

### Transformers

Understand:

* Tokens
* Embeddings
* Attention
* Context Window

Resource:

* The Illustrated Transformer

---

### Prompt Engineering

Topics:

* System prompts
* Structured outputs
* Few-shot prompting

Goal:

Reliable LLM responses.
