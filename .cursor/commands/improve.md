# Improve Command

Analyze the selected code and suggest improvements.

## Instructions

Follow:

- DRY
- SOLID
- KISS
- PEP8

Preserve behavior.

Do not suggest unnecessary abstractions.

Focus on:

- Readability
- Maintainability
- Reusability
- Simplicity

## Output Format

1. **Current Issues**
2. **Suggested Improvements**
3. **Expected Benefits**
4. **Potential Risks**
