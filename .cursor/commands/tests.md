# Tests Command

Generate production-quality tests for the selected code.

## Instructions

Create tests for:

- Happy path
- Edge cases
- Invalid input
- Error handling
- Permission checks
- Business rules

Reuse existing fixtures when possible.

Follow existing project testing conventions.

Do not modify production code.

## Output Format

1. **Testing Strategy**
2. **Test Cases**
3. **Test Implementation**
4. **Coverage Notes**
