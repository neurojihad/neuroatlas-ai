# Architecture Command

Analyze the selected code, module, service, or feature from an architectural perspective.

## Instructions

Review:

- Layer boundaries
- Dependency flow
- Separation of concerns
- SOLID compliance
- Clean Architecture compliance
- Scalability
- Maintainability

Respect project architecture:

api -> services -> repositories -> database

## Output Format

1. **Current Architecture**
2. **Strengths**
3. **Weaknesses**
4. **Architecture Risks**
5. **Recommendations**
