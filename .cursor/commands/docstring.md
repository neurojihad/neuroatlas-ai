# Docstring Command

Generate docstrings for the selected code.

## Instructions

- Add class-level docstrings.
- Add method docstrings.
- Describe purpose.
- Describe parameters.
- Describe return values.
- Describe raised exceptions.
- Follow project documentation standards defined in project rules.

## Output Format

Return updated code with docstrings added.
