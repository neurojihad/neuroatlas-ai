# Optimize Command

Analyze the selected code for performance issues.

## Instructions

Review:

- Database queries
- N+1 query problems
- Memory usage
- Loops and iterations
- Network calls
- Async usage
- Caching opportunities

Prefer measurable improvements.

Avoid premature optimization.

## Output Format

1. **Performance Bottlenecks**
2. **Impact Analysis**
3. **Recommended Optimizations**
4. **Expected Benefits**
5. **Potential Trade-offs**
