# Debug Command

Investigate the selected code, logs, stack trace, or error.

## Instructions

- Analyze symptoms.
- Trace execution flow.
- Identify likely root causes.
- Rank hypotheses by probability.
- Explain reasoning.
- Recommend the smallest safe fix.
- Avoid speculative fixes.

## Output Format

1. **Observed Issue**
2. **Investigation Summary**
3. **Possible Causes**
4. **Most Likely Root Cause**
5. **Recommended Fix**
6. **Potential Side Effects**
